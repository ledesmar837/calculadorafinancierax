# CalculadoraFinancieraX

**Tu calculadora financiera personal. Sin registro, sin cuentas. Todo privado en tu navegador.**

![Next.js](https://img.shields.io/badge/Next.js-16-black) ![TypeScript](https://img.shields.io/badge/TypeScript-5-blue) ![Dark](https://img.shields.io/badge/Theme-Dark-818cf8)

---

### 🚀 Demo

[https://calculadorafinancierax.vercel.app](https://calculadorafinancierax.vercel.app)

### ✨ Funcionalidades

- Calculadora financiera con 20 campos de gastos
- Gráficos interactivos (Pie + Bar)
- Informe personalizado con insights automáticos
- Comparación con regla 50/30/20
- Fondo de emergencia estimado
- Blog con 8 artículos educativos
- Modo oscuro/claro
- Multi-moneda (COP, USD, EUR, MXN, ARS, CLP, PEN, BRL)
- Verificación anti-bot (Cloudflare Turnstile)
- 100% localStorage — sin backend, sin datos personales

### 🛠️ Tecnología

Next.js 16, TypeScript, Tailwind CSS 4, Framer Motion, Recharts

### 🧪 Desarrollo

```bash
npm install
npm run dev     # localhost:3000
npm run build   # build producción
```

### 🌐 Despliegue

1. Sube el repo a GitHub
2. Conecta con [Vercel](https://vercel.com/new) (plan gratuito)
3. Para dominio `calculadorafinancierax.is-a.dev`:
   - Sigue el proceso de PR en [is-a.dev](https://is-a.dev)
   - Usa `CNAME: calculadorafinancierax.vercel.app`
   - Configura el dominio en Vercel → Settings → Domains
